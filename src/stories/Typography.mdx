# Tipografi

Sistem tipografi untuk PT Sarana Pancakarya Nusa dirancang untuk menciptakan hierarki visual yang jelas dan meningkatkan keterbacaan di seluruh aplikasi.

## Font Family

Aplikasi menggunakan font Poppins sebagai font utama untuk semua elemen UI. Font ini dipilih karena keterbacaannya yang baik dan tampilan yang modern.

```jsx
// Definisi font di src/app/theme.ts
export const poppins = Poppins({
  weight: ['300', '400', '500', '600', '700'],
  subsets: ['latin'],
  display: 'swap',
});
```

## Penggunaan Tipografi

### Penggunaan Heading

- **Heading 1**: Digunakan untuk judul utama halaman.
- **Heading 2**: Digunakan untuk judul bagian utama.
- **Heading 3-6**: Digunakan untuk sub-judul dengan tingkat kepentingan yang menurun.

### Penggunaan Body Text

- **Body 1**: Digunakan untuk teks konten utama.
- **Body 2**: Digunakan untuk teks konten sekunder atau pendukung.

### Penggunaan Utilitas

- **Button**: Digunakan untuk teks pada tombol.
- **Caption**: Digunakan untuk keterangan, label kecil, atau teks bantuan.
- **Overline**: Digunakan untuk label kategori atau header kecil di atas konten.

## Responsivitas

Sistem tipografi dirancang untuk responsif di berbagai ukuran layar. Font size akan menyesuaikan secara otomatis pada breakpoint yang berbeda menggunakan `responsiveFontSizes` dari Material UI.

```jsx
// Implementasi responsiveFontSizes di src/app/theme.ts
let theme = createTheme(themeOptions);
theme = responsiveFontSizes(theme);
```

## Praktik Terbaik

- Gunakan hierarki tipografi yang konsisten di seluruh aplikasi.
- Batasi jumlah ukuran font yang digunakan untuk menjaga konsistensi.
- Pastikan kontras warna teks dan latar belakang memenuhi standar aksesibilitas WCAG 2.0 AA.
- Gunakan line height yang cukup untuk meningkatkan keterbacaan, terutama untuk teks yang panjang.
- Hindari penggunaan terlalu banyak variasi font weight dalam satu halaman.
