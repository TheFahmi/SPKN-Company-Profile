# Spacing dan Shadows

Sistem spacing dan shadows untuk PT Sarana Pancakarya Nusa dirancang untuk menciptakan tata letak yang konsisten dan memberikan kedalaman visual pada elemen UI.

## Spacing

Sistem spacing menggunakan basis 8px dengan kelipatan yang digunakan secara konsisten di seluruh aplikasi. Ini membantu menciptakan ritme visual dan keselarasan antar elemen.

### Basis Spacing

```jsx
// Definisi spacing di src/app/theme.ts
const spacing = 8;
```

### Penggunaan Spacing

Spacing digunakan untuk:

- **Margin**: Jarak antara elemen-elemen UI.
- **Padding**: Jarak internal dalam elemen UI.
- **Gap**: Jarak antara item dalam grid atau flex container.

```jsx
// Contoh penggunaan spacing dalam komponen
<Box sx={{ p: 2 }}>                  {/* padding: 16px (2 * 8px) */}
<Box sx={{ m: 3 }}>                  {/* margin: 24px (3 * 8px) */}
<Box sx={{ px: 2, py: 3 }}>          {/* padding-left & right: 16px, padding-top & bottom: 24px */}
<Box sx={{ mt: 2, mb: 4 }}>          {/* margin-top: 16px, margin-bottom: 32px */}
<Stack spacing={2}>                  {/* gap: 16px antara item */}
```

## Border Radius

Border radius digunakan untuk memberikan sudut melengkung pada elemen UI, menciptakan tampilan yang lebih lembut dan modern.

### Penggunaan Border Radius

- **Small (4px)**: Digunakan untuk elemen kecil seperti chip, badge, dan tooltip.
- **Medium (8px)**: Digunakan untuk tombol, input, dan elemen UI umum lainnya.
- **Large (12px)**: Digunakan untuk kartu, dialog, dan panel.
- **Extra Large (16px)**: Digunakan untuk kartu yang lebih besar atau elemen yang memerlukan sudut yang lebih lembut.
- **Round (50%)**: Digunakan untuk avatar, ikon bulat, dan elemen yang memerlukan bentuk lingkaran.

- **Elevation 16**: Dialog dan modal yang memerlukan fokus pengguna.
- **Elevation 24**: Drawer dan elemen yang sangat menonjol.

## Mode Gelap

Dalam mode gelap, shadows dinonaktifkan untuk mengurangi kontras visual yang berlebihan dan menciptakan tampilan yang lebih nyaman di lingkungan cahaya rendah.

```jsx
// Implementasi shadows berdasarkan mode di src/app/theme.ts
if (mode === 'light') {
  themeOptions.shadows = shadows;
} else {
  themeOptions.shadows = Array(25).fill('none');
}
```

## Praktik Terbaik

- Gunakan spacing yang konsisten untuk menciptakan tata letak yang harmonis.
- Gunakan border radius yang sesuai dengan ukuran dan fungsi elemen.
- Gunakan shadows untuk memberikan kedalaman dan hierarki pada elemen UI.
- Hindari penggunaan shadows yang terlalu banyak atau terlalu kuat, karena dapat mengganggu fokus pengguna.
- Pastikan elemen dengan elevation yang lebih tinggi memiliki shadow yang lebih kuat untuk menciptakan hierarki visual yang jelas.
