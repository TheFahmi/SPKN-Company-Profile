## Penggunaan Warna

### Warna Teks

- **Teks Primer**: #212121 (Grey 900) - Digunakan untuk teks utama seperti judul dan konten.
- **Teks Sekunder**: #757575 (Grey 600) - Digunakan untuk teks pendukung dan keterangan.
- **Teks Dinonaktifkan**: #9e9e9e (Grey 500) - Digunakan untuk teks yang tidak aktif atau dinonaktifkan.

### Warna Latar Belakang

- **Latar Belakang Default**: #ffffff - Digunakan untuk latar belakang halaman.
- **Latar Belakang Paper**: #ffffff - Digunakan untuk kartu, dialog, dan elemen UI lainnya.
- **Latar Belakang Terang**: #f5f5f5 (Grey 100) - Digunakan untuk area yang membutuhkan perbedaan visual ringan.

### Warna Divider

- **Divider**: rgba(0, 0, 0, 0.12) - Digunakan untuk garis pemisah antar elemen.

## Panduan Penggunaan

- Gunakan warna utama untuk elemen-elemen interaktif utama seperti tombol dan link.
- Gunakan warna status untuk memberikan umpan balik visual kepada pengguna.
- Gunakan warna abu-abu untuk teks dan elemen-elemen UI lainnya yang membutuhkan nuansa netral.
- Pastikan kontras warna memenuhi standar aksesibilitas WCAG 2.0 AA (rasio kontras minimal 4.5:1 untuk teks normal dan 3:1 untuk teks besar).
- Gunakan warna secara konsisten di seluruh aplikasi untuk menciptakan pengalaman pengguna yang kohesif.
